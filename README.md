# CloudOps-Academy-Automated-AWS-CI-CD-Pipeline

CloudOps Academy is a modern, 4-tier microservice web application designed to demonstrate end-to-end Infrastructure as Code (IaC) and continuous deployment practices. Built with a React frontend, a Node.js/Express backend, and a PostgreSQL database, the application is securely routed through an Nginx reverse proxy.

Rather than a manual deployment, the entire stack is containerized with Docker, dynamically provisioned on AWS EC2 using Terraform, configured seamlessly via Ansible, and automatically deployed through a GitHub Actions CI/CD pipeline. This project serves as a comprehensive blueprint for scalable cloud architecture and automated operations.
